Anti-India Campaign Detection Dashboard

View Project on GitHub

Problem Statement

Develop a system to detect anti-India campaigns on digital platforms by building a keyword database and using engagement analysis to identify key drivers, supporting early detection.

Description

The proposed system combines keyword-based monitoring with engagement analysis to identify anti-India campaigns. A dynamic database of flagged terms, phrases, and hashtags is created and updated. Using AI and natural language processing (NLP), the system scans social media, forums, and websites to detect content matching this database. Engagement metrics like likes, shares, comments, and network reach are then analyzed to identify influential accounts and viral trends. This enables early detection of coordinated disinformation or propaganda efforts, supporting timely countermeasures by security agencies and helping preserve national integrity in the digital space.

Key Tasks

  • Build and maintain a dynamic keyword, hashtag, and phrase database linked to anti-India sentiments.
  • Develop real-time monitoring tools to scan social media, forums, and websites for flagged content.
  • Implement NLP techniques to filter and classify relevant anti-India content at scale.
  • Conduct engagement analysis on likes, shares, comments, and reach to track viral content.
  • Identify key influencers, pages, or groups spreading harmful content and map their networks.
  • Detect coordinated campaigns based on synchronized posting or hashtag usage.
  • Set up an alert system for trending narratives and high-engagement propaganda.
  • Design a visualization dashboard for flagged content, engagement data, and influencer networks.
  • Ensure compliance with free speech and privacy laws while monitoring digital platforms.
  • Provide actionable reports to government and security agencies for preventive countermeasures.

Tech Stack

  • Backend: Python (Flask)
  • Frontend: HTML, TailwindCSS, JavaScript
  • Data Processing: NLP models, Engagement Analysis
  • Visualization: Plotly, Chart.js
