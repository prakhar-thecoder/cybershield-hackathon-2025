Problem Statement

Social media platforms are increasingly exploited to spread disinformation and anti-national propaganda. Manual detection of such campaigns is slow, inefficient, and often misses fast-spreading harmful content. There is a strong need for an automated system that can detect, analyze, and report anti-India campaigns in real-time.

Our Solution

Threat Hunters is a prototype monitoring system that scrapes social media data, processes it using Natural Language Processing (NLP), and identifies anti-India sentiments. It provides real-time dashboards, generates automated reports, and delivers insights directly to users‚Äô emails to assist cybersecurity teams and law enforcement agencies in tracking harmful campaigns.

Key Features

  • Hashtag-based monitoring for targeted campaign detection
  • Scraping and analysis of posts from social media platforms
  • AI-powered sentiment classification for detecting anti-India content
  • Interactive dashboard with real-time statistics and visualizations
  • Automated PDF report generation with email delivery
  • Prototype optimized for hackathon-scale limited resources

Tech Stack

  • Backend: Python (Flask)
  • Frontend: HTML, TailwindCSS, JavaScript
  • Data Processing: NLP models, Sentiment Analysis
  • Visualization: Plotly, Chart.js
